Projects

Meduusa Time Tracker

Spring Boot, Bootstrap, Javascript12/2016
Group project made during a software development course in Haaga-Helia. Our goal was to create a time tracker web application that would let users log how many hours they have spent working on a specific project. I wrote most of the Java backend and everything database related. The Github repository has a better list of all the implemented features and technologies used.

The application itself is up and running here and can be accessed using credentials demo / salasana. You can also create a new account if you wish.

Työkykypassi

Java, Spring, Android, Bootstrap01/2017
Application that I'm currently working on during my internship in Haaga-Helia University of Applied Sciences. It's an application for enhancing well-being, occupational health and environment of professionals in different areas.
The application consists of an Android application, a REST backend made with Java and Spring Framework, and a web application for administrators. The program is split into three different repositories in Github.

Development was started in early 2016 as a part of a programming course. I was later hired to finish the program, as the previous developers finished the course. My responsibilities include full stack development in all parts of the application, testing, documentation, setting up the production server, and application deployment.
